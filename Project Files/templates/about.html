About Smart Sorting

Empowering freshness with AI

NutriGaze is an AI-powered solution built using Transfer Learning to classify fruits and vegetables as fresh or rotten. This smart sorting system uses a deep learning model trained on real-world produce images and enables instant quality detection via an easy-to-use web interface.

  • 🔍 High accuracy fruit/vegetable quality detection
  • 🚀 Powered by MobileNetV2 transfer learning
  • 🌐 Deployed with Flask for real-time access
  • 📦 Helps reduce food waste in supply chains

Developed as an AIML domain project to combine real-world impact with cutting-edge technology.
